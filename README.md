#  Earth Facts and Information (HTML)

**Earth Facts and Information** is a simple HTML-based project that presents interesting facts and essential details about our planet Earth.  
It is created as a practice project to understand and apply basic HTML structure and elements.

---

##  Project Overview

This project showcases key facts and information about Earth using basic HTML elements. It includes:

- Introductory text
- Lists of facts
- A comparison table with other planets
- Multimedia content (image and video)
- External links for further reading

---

##  Features

- **Introduction Section**
  - Displays a welcoming message and brief description of Earth

- **Basic Facts**
  - Polar diameter
  - Mass
  - Number of moons (with external link)

- **Top 3 Facts**
  - Presence of liquid water
  - Extremely hot core
  - Nitrogen-rich atmosphere

- **Planet Comparison Table**
  - Compares Earth with:
    - Mars
    - Venus
  - Includes radius and average temperature

- **External Link**
  - Button that redirects users to Earthâ€™s Wikipedia page

- **Embedded Video**
  - Video content related to Earth

##  How to Run the Project

1. Clone the repository or download the files.

   - `index.html`
   - `Earth Image.jpg`
   - `Earth Video.mp4`
3. Open `index.html` in any web browser:
